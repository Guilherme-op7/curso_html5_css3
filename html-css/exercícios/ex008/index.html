Itálico / ênfase

nesta frase temos um termo em itálico usando a tag <I> (Não semântica)

também temos uma parte marcada para dar ênfase com a tag <EM>

Marca de texto

neste parágrafo temos esta frase em destaque usando a tag <MARK>

Letras Grandes

nesta frase temos uma parte com Letras Grandes usando a tag <BIG>

letras pequenas

nesta frase temos esta parte muito pequena usando a tag <SMALL>

texto deletado

podemos marcar um texto excluído para indicar que ele deve ser lido mas não considerado. usando a tag <DEL>

texto inserido

também podemos marcar um texto inserido para indicar que ele foi adicionado depois e dar ênfase. usando a tag <ins>

existe também o sublinhado com a tag<U> (não semântica)

texto sobrescrito

para marcamos coisas como x2+13 usando a tag <sup>

texto subscrito

para marcarmos coisas como H2O usando a tag <SUB>
